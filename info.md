# Automation Map — HACS info

## 🗺️ Automation Map

Interaktive Prozesslandkarte für alle Home Assistant Automatisierungen, Entitäten und Helfer.

**Features:**
- Interaktiver Graph mit Knoten & Kanten
- Echtzeit-Werte via WebSocket
- Automatische Gruppierung nach HA-Bereichen
- Detail-Panel mit Trigger/Bedingungen/Aktionen
- Navigation zum HA-Editor
- Adaptives Theme (hell/dunkel)
